# SwagShop
De applicatie is een fake webshop waar mensen Brightest promo spullen op kunnen kopen. Deze webshop heeft opzettelijk bugs ingebouwd die aan en uitgezet kunnen worden vanuit het adminpaneel. Dit laat ons toe om test automation te laten zien aan bijvoorbeeld klanten op objecten die zich niet goed gedragen om de kracht van test automation te tonen.
